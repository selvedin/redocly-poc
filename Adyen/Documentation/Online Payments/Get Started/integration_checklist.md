# Integration Checklist


## 1. Set Up Your Test Account
1. **Create your account**
2. **Determine testing account structure**
3. **Create users for team members**
4. **Get API credentials**
5. **Add payment methods**

---

## 2. Build Your Integration
1. **Set up webhooks**
2. **Build the integration**
3. **Handle errors and refused payments**
4. **Modify payments**
5. **Add additional payment methods**
6. **Complete your integration with additional features**
    - Consider implementing:
        - Tokenization
        - Account Updater
        - Auto Rescue
        - Donations
        - Network tokenization
        - Partial payments
        - 3D Secure 2 authentication (for fraud protection and PSD2 SCA compliance)
7. **Test your integration**
    - Use the testing guide to verify all transaction types your integration supports.

---

## 3. Reporting and Reconciliation
1. **Download the Settlement details report**
2. **Build reconciliation processes**

---

## 4. Go Live

- configure live settings
  - account
  - finance
  - risk and compliance
  - API communication
  - webhooks
- end to end testing
- capture
- redirect payment methods
- 3D secure
- tokenization
